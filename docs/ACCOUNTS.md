# テスト用アカウント

ローカル開発・動作確認で利用するテストアカウントの一覧です。**本番環境では使用しません。**

## アカウント一覧

| ロール | メールアドレス | パスワード | 用途 |
| :--- | :--- | :--- | :--- |
| 管理者 | `admin@mail.com` | `admin1234` | 管理者向け画面・機能の動作確認 |
| 開発者 | `dev@mail.com` | `dev1234` | 開発者向け画面・機能の動作確認 |
| エンドユーザー | `user@mail.com` | `user1234` | 一般ユーザー向け画面・機能の動作確認 |

## ログイン手順

1. アプリをローカルで起動する（[README.md](../README.md#アプリの起動) 参照）。
2. フロントエンド（[http://localhost:3000/](http://localhost:3000/)）のログイン画面を開く。
3. 上記いずれかのメールアドレスとパスワードを入力してログインする。

ログイン後の判定ロジックは [`client/src/lib/adminRole.ts`](../client/src/lib/adminRole.ts) を参照してください。

## 注意事項

- 本ドキュメントに記載されたアカウントは **ローカル開発・検証専用** です。本番環境のアカウント情報は記載しません。
- パスワードを変更する場合は [`client/src/lib/adminRole.ts`](../client/src/lib/adminRole.ts) と本ドキュメントの両方を更新してください。
